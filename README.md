# EV3 Robot Project

##  Project Overview
This project involves developing an EV3 robot capable of line following and obstacle detection. The goal is to simulate a real software development workflow using Agile practices and collaborative tools.

##  Objectives
- Build a robot that can follow a line using a light sensor  
- Implement obstacle detection using an ultrasonic sensor  
- Apply Agile project management practices  
- Use version control for collaboration  

##  Tools Used
- Jira – Task management and sprint tracking  
- Confluence – Project documentation and meeting notes  
- GitHub – Version control and collaboration  

## Team Members
- Imalka Siriwardena  
- Emil Lahtinen  

## Workflow Summary
The project was developed using an Agile Scrum approach:
- Tasks were created and managed in Jira  
- Meetings and documentation were recorded in Confluence  
- Code and project files were managed using GitHub with commits and collaboration  

## Features
- Line-following movement system  
- Obstacle detection system  
- Basic sensor calibration and testing  
- Iterative debugging and improvement process  

## Project Status
This project is in development and demonstrates a complete Agile workflow from planning to implementation and review.

## Learning Outcomes
- Understanding of Agile Scrum workflow  
- Experience with Jira, Confluence, and GitHub integration  
- Basic robotics logic using EV3 system  
- Team collaboration and task distribution
